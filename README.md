# Edge-TTS 多角色音频生成器

一个基于 PyQt6 和 Edge-TTS 的多角色音频生成工具，支持角色切换、停顿控制和蜂鸣声功能。

## 功能特点

1. **多角色切换**：使用 [A]、[B]、[C]、[D] 标记切换角色
2. **停顿控制**：使用 [数字] 标记添加停顿（单位：毫秒）
3. **蜂鸣声**：使用 [R] 标记添加蜂鸣声
4. **独立角色设置**：每个角色可单独设置发音人、语速、音量、语调
5. **试听功能**：支持试听选中文本或全部文本
6. **音频生成**：分段生成音频并合并成完整文件
